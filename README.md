# cursor-setup-project
Junior Growth Marketing Specialist Portfolio Project
## 1. Tools Installed
- **Cursor IDE**
- **Claude Code** (Extension)
- **Codex** (OpenAI’s coding agent)

## 2. Completed Steps
1. Installed Cursor IDE.
2. Configured AI extensions (Claude Code & Codex).
3. Created a public GitHub repository.
4. Set up the local project environment.
5. Documented the process in this README file.
6. Committed and pushed the changes to the GitHub repository.

## 3. Troubleshooting & Problem Solving
During the setup process, I encountered a few technical challenges:

* **Codex Extension:** The automatic installation via command line failed initially. I diagnosed that the system required the unique Extension ID (`publisher.name`) rather than the descriptive name. I successfully installed it via the Cursor Marketplace GUI.
* **Git Configuration:** I faced an error when trying to use `git` commands in the terminal because Git was not yet installed on my system. To maintain efficiency and meet the 48-hour deadline, I opted for a contingency approach: I downloaded the repository as a ZIP file, worked on the files locally, and uploaded the updates directly through the GitHub web interface. This allowed me to continue progressing without project delays.
